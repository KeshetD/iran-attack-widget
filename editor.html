3. ערוך את הקובץ לחץ על עיפרון ✏️ לעריכה ← מצא את const DATA = { ← שנה את pct1 ו-pct2

🚀 פרסום ראשוני ב-GitHub Pages

1. צור חשבון GitHub אם אין לך — הירשם בחינם ב-github.com
2. צור Repository חדש לחץ New repository ← תן שם (לדוג' iran-attack-widget) ← Public ← Create
3. העלה את הקבצים לחץ Add file → Upload files ← גרור את index.html ו-editor.html ← Commit
4. הפעל GitHub Pages Settings ← Pages ← Source: main ← Save
הלינק יהיה: https://[שם-משתמש].github.io/[שם-repo]/ ✓ האתר חי!
💡 טיפ: שתף את הלינק הראשי עם כולם, ואת /editor.html רק עם העורכים.
